The ROOT Project

ROOT is a framework for data processing, born at CERN, at the heart of the research on high-energy physics. Every day, thousands of physicists use ROOT applications to analyze their data or to perform simulations

  • More

JavaScript ROOT

JavaScript ROOT aims to provide interactive ROOT-like graphics in web browsers. One could read and display data from binary ROOT files. Or data can be obtained from JSON representation, produced with TBufferJSON class.
